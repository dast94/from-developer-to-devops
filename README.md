# From Developer to DevOps

## About me

I am a Software Developer with around 3 years of professional experience.

I have a Bachelor's degree in Computer Engineering and a Master's degree in Telecommunications Engineering.

My goal is to move from Software Development toward DevOps, Cloud Engineering, Platform Engineering, or SRE roles.

## Goal

The goal of this repository is to document my learning path from Software Developer to DevOps Engineer.

This is not just a collection of notes.

Each module will include:

- essential theory
- practical labs
- exercises
- troubleshooting notes
- final challenges
- interview preparation

## Roadmap

1. Linux
2. Git
3. Docker
4. Docker Compose
5. Networking
6. GitHub Actions
7. AWS
8. Terraform
9. Kubernetes
10. Monitoring
11. Logging
12. Security
13. Final Project

## Method

The focus is on practical learning.

Every topic must produce something concrete:

- commands
- scripts
- configuration files
- documentation
- debugging notes
- real-world examples

## Final objective

At the end of this journey, I want to be able to explain, build, debug, and operate a modern cloud-native infrastructure.
